Like Me – Desafío Full Stack (Parte I)

Este proyecto corresponde al desarrollo del backend de una red social llamada **“Like Me”**, enfocada en la creación de publicaciones y la interacción de likes entre los usuarios.

El objetivo fue construir un servidor con **Node.js** y **Express**, conectado a una base de datos **PostgreSQL** mediante el paquete **pg**.

Características principales

- Se habilitó **CORS** para permitir la comunicación entre el frontend (aplicación React) y el backend.
- Conexión con la base de datos **PostgreSQL** llamada `likeme`.
- Creación de una ruta **GET** para obtener todos los posts almacenados en la base de datos.
- Creación de una ruta **POST** para agregar nuevos posts con título, imagen, descripción y likes.
- Este proyecto representa la primera etapa del desarrollo de una red social con funcionalidades de interacción entre usuarios.

Tecnologías utilizadas

- Node.js
- Express
- PostgreSQL
- pg
- CORS
- dotenv

Like Me – Full Stack Challenge (Part I)

This project is the backend development of a social network called **“Like Me”**, focused on creating user posts and managing likes.

The goal was to build a server using **Node.js** and **Express**, connected to a **PostgreSQL** database via the **pg** package.

Key Features

- **CORS** enabled to allow communication between the frontend (React application) and the backend.
- Connected to a **PostgreSQL** database named `likeme`.
- Created a **GET** route to fetch all posts stored in the database.
- Created a **POST** route to add new posts with title, image, description, and likes.
- This project represents the first stage in developing a social network with user interaction functionality.

Technologies Used

- Node.js
- Express
- PostgreSQL
- pg
- CORS
- dotenv
